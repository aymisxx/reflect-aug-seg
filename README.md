# **Reflect-Aug-Seg** 📡🫧

**(Reflectivity-Augmented LiDAR Semantic Segmentation for Robotic Perception)**

## Overview

This project explores whether **reflectivity-like features derived from LiDAR intensity** can improve semantic understanding and segmentation-oriented perception in robotic systems.

Instead of relying purely on geometry and raw intensity, it introduces a **pseudo-reflectivity feature** that compensates for distance-dependent distortions in LiDAR measurements.
